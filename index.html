Application Instructions


User insructions/Use

To start creating an order the user must first go to the products page by clicking the products button at the top. On the products page the user can view the pub menu on whats for sale.


The user can then go back off the menu and select the product they want with the price and quantity of the product. The user will only be able to input the correct price of £1 for the products as it is a £1 shop. Once correct details Have been entered the user will click create button. This will add the product to the basket/orderdetails and take them to the orderdetails page. The user can then decide on adding another product to the order by clicking the add another product to our order button.


The user also has the choice to cancel their order for a item by clicking the cancel order link. If the user is happy with their order then the complete order can be clicked and the user will have to select their table and click the create button. This will bring the user to a page saying thank you and it shows their table and order date.


The user can click back on orderdetails to see the orderstatus on their order has changed to complete(tick) and orderID has been given.

Admin Instructions/Use

The admin can access the admin page by clicking the admin button on the top of the page.


The admin will be brought to the main admin page where they can see all the products and have the choice to edit a product, delete a product, create a new product and also view customers order details.


To add or edit a product all the admin has to do is click the corresponding link and type into the following input boxes the correct details matching the box lable and then click the save or create button at the bottom depending on activity. If the admin clicks the view customer details button the admin will be able to see all the customers order details in table.
